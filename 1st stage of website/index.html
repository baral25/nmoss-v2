Nmoss Network monitor is simple & lightweight yet it boasts a variety of features that should
aid users to garner a better understanding of their networks.
Some key features of Nmoss are listed below:


  • -Tool Completely made in Python

  • -Non_Intrusive

  • -Creates logs after each execution

  • -Web Application for UI experience

  • -PRivacy of Data

  • -Lightweight

    • About.

    Nmoss Network Monitor is part of an ongoing project. As the project is conceptualized, developed and executed solo, it has follws the same ethos of Privacy of data & ease of se. The tool is extremely lightweight (written in Python) and is based mostly around Command Prompt (CMD). After execution the tool creates a text log file that documents all relevant activites in the network, this information can be hard to interpret for newer users. To facilitate the inexperience users the File Reader offers a more manged view of the network information. Also the File Reader is completely web based and doesn't store any of the network information of the user. With
